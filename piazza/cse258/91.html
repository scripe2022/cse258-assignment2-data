You're welcome to use the ratings if you can find a way that they're useful. E.g. if you're trying to predict what they'll read in the future, previously read items might only be relevant if they actually liked the previous items. So you could e.g. incorporate ratings into a weighted voting function (or something like that).

\n

\n

Generally speaking, it should be impossible to "cheat" if you don't use anything external to the provided data. That being said, I don't remember people getting a lot of value out of rating data for the "would read" task in the past (though this year's data could be different!)
