My personal tip is, once you have successfully set up the running environment:

\n

\n
  1. You can start from running and reading the codes (and the output of the codes). The codes on the stub file already help you read the dataset. Getting familiar with the dataset is an important first step.
  2. We cannot directly use the dataset to train the model because it contains quite a lot of information — not all of them are helpful in the problem. You need to think about what information is useful and then try to derive the features (X) and the labels (Y) from the dataset. Different homework questions may ask you to extract different features and labels.
  3. Implement adequate model to solve the problem. For linear regression, you can refer to numpy.linalg.lstsq. The documentation of the function tells you what is the parameters of the function and what it returns along with a few examples. Replace the data in their examples with your own data can help you get started quickly.
  4. Once you get the solution to the regression problem, besure to format your output accordingly.
